# 🏦 Sistema Bancário em Python

Um projeto simples desenvolvido em **Python** que simula operações
básicas de um sistema bancário via terminal.

Este projeto foi criado com o objetivo de praticar **Programação
Orientada a Objetos (POO)** e conceitos básicos de Git/GitHub.

------------------------------------------------------------------------

## 📌 Funcionalidades

O sistema permite:

-   ✅ Cadastro de cliente (CPF, nome e saldo inicial)
-   💵 Saque
-   💰 Depósito
-   🏦 Aplicação em poupança
-   📉 Pagamento de parcela de empréstimo
-   ⚡ PIX depósito
-   🔁 PIX transferência
-   📊 Atualização automática do saldo após cada operação
-   📋 Menu interativo no terminal

------------------------------------------------------------------------

## 🛠 Tecnologias Utilizadas

-   Python 3
-   Git
-   GitHub
-   VS Code

## 🚀 Como Executar o Projeto

### 1️⃣ Clone o repositório

git clone https://github.com/weslley0gg/sistema-bancario-python.git

### 2️⃣ Acesse a pasta do projeto

cd sistema-bancario-python

### 3️⃣ Execute o programa

python nome_do_arquivo.py

(Substitua `nome_do_arquivo.py` pelo nome real do seu arquivo.)

------------------------------------------------------------------------

## 💡 Exemplo de Uso

Ao executar o programa, o usuário poderá:

1.  Informar CPF
2.  Informar nome
3.  Informar saldo inicial
4.  Escolher operações através de um menu interativo

Todas as operações mostram o saldo atualizado após execução.
